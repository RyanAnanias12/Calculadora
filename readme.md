# 📟 Calculator (C# Console)

Uma calculadora simples feita em **C#** para realizar operações matemáticas básicas no terminal.  

## 🚀 Funcionalidades
- ➕ **Soma**
- ➖ **Subtração**
- ✖ **Multiplicação**
- ➗ **Divisão**
- 🔄 Menu interativo no console
- ❌ Opção de sair do programa
